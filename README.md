# 速窺運勢

八字 × 易經 × 奇門遁甲，每日時辰推演工具

**線上體驗**：https://donma.github.io/PeekFate/

---

## 專案介紹

**速窺運勢**是一個純前端命術規則引擎 PWA，不儲存個資，可離線使用。

使用者進入網站後輸入西元生日，出生時間選填。系統直接在瀏覽器用 JSON 規則資料庫與 JavaScript 推算，輸出今日與明日每個時辰的吉凶、可能事件、適合事項、避免事項與白話建議，並提供接下來 14 天的每日概述。

### 核心價值

- 不登入、不上傳生日、不保存個資
- 可離線使用、可安裝到手機主畫面
- 所有結果由規則推算，每個判斷可追溯來源
- 今日 + 明日 24 時辰為主軸，接下來 14 天為輔助概述
- 支援性別欄位，自動推算大運（陽男陰女順排，陰男陽女逆排）
- 節氣數據庫涵蓋 1900–2100 年，月柱精準切換

## 技術架構

### 前端技術

- HTML5
- CSS3（RWD 響應式設計）
- JavaScript ES6+
- PWA（Progressive Web App）

### 命術系統

- **八字**：四柱推算、十神、地支關係、藏干、納音、大運（需性別）
- **五行**：相生相剋、元素屬性、旺相休囚死簡化分析
- **易經 64 卦**：卦象、變卦、互卦、錯卦、綜卦、五行個人化
- **奇門遁甲時家盤**：九宮、八門、九星、八神、格局判讀

## 功能特色

### 1. 性別與大運

新增性別欄位，系統自動推算陽男陰女順排／陰男陽女逆排大運，每十年一運共八步，並納入計分系統。

### 2. 廣域節氣支援

節氣數據擴充至 1900–2100 年（201 年 × 24 節氣），使用天文演算法 + UTC+8 校正，月柱換節精準。

### 3. 自動儲存

輸入過的生日資料會自動儲存在 localStorage，下次開啟自動載入並推算。

### 4. 時辰趨勢圖

今日/明日時辰以長條圖顯示，一眼看出運勢高低。

### 5. 環形分數圖

今日/明日總覽以環形圖顯示平均分數。

### 6. 14 天趨勢圖

接下來 14 天以連續長條圖顯示運勢趨勢。

### 7. 白話解釋

所有計分來源都以白話文顯示，用戶看得懂。

### 8. 收合表單

推算完成後表單自動收合，直接顯示結果。

## 計分規則

### 總分計算

```
總分 = 基礎分數(60) + 八字分數 + 易經分數 + 奇門分數 + 平衡分數
```

### 分數範圍

| 來源 | 範圍 |
|------|------|
| 基礎分數 | 60 |
| 八字 | ±25 |
| 易經 | ±20 |
| 奇門 | ±35 |
| 平衡 | ±10 |

### 吉凶等級

| 分數 | 等級 | 說明 |
|------|------|------|
| 85-100 | 大吉 | 諸事順遂，大吉大利 |
| 70-84 | 吉 | 順利吉祥，宜積極行動 |
| 55-69 | 小吉 | 較為順利，可適度進取 |
| 45-54 | 平 | 平穩無波，維持現狀為宜 |
| 30-44 | 小凶 | 略有阻礙，謹慎行事 |
| 0-29 | 凶 | 諸事不順，宜靜不宜動 |

## 已知限制

### 八字流派

- 子時跨日規則固定為 23:00 換日，少數流派可能不同
- 節氣數據使用天文演算法，偶有 ±1 日誤差（影響月柱邊界日出生者）

### 奇門遁甲

- 流派繁多，本系統採固定時家奇門規則

### 大運

- 性別為選填，未填則不推算大運
- 大運吉凶判定為簡化版（比和/生我為吉），可後續深化

### 命術結果

- 屬民俗文化參考
- 不作為醫療、法律、投資、婚姻或人生重大決策保證
